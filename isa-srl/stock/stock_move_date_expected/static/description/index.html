Stock Move Date Expected

Funzioni principali del modulo

Questo modulo permette di impostare la data pianificata scelta, per un picking; la seguente modifica é situata sotto: "Magazzino->Tutte le operazioni".
Durante la creazione di un trasferimento, si puó scegliere di impostare la data pianificata.

Nel caso in cui viene impostata, vengono settate automaticamente, le seguenti variabili del picking:
  • Data
  • Data pianificata

    • Altrimenti, se non viene impostata la data pianificata, prenderá di default, la data odierna.

    Inoltre, quando viene trasferita la merce da un magazzino all'altro, nelle quants é stato modificato il campo "incoming_date", in cui viene impostata la data pianificata inizialmente.
