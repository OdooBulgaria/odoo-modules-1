Sale Order Lot

Introduce all'interno dell'ordine di vendita i lotti di produzione (se in configurazione ne é impostato l'utilizzo)

Durante la creazione di un preventivo/ordine di vendita, se in configurazione magazzino é impostata la gestione dei lotti di produzione, viene data la possibilitá di specificare il lotto.

Il lotto, presente sulle righe dell'ordine di vendita, viene poi riportato sul picking e durante il trasferimento viene proposto in automatico nelle righe del transfer. Le righe possono comunque essere modificate.
